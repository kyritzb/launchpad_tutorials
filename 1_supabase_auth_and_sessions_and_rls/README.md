# Supabase Auth, Sessions & Row-Level Security

Next.js project demonstrating Supabase authentication, session management, and row-level security (RLS) policies.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- [Supabase](https://supabase.com) (Auth + Postgres + RLS)
- TypeScript
- Tailwind CSS

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Copy the environment variables and fill in your Supabase project credentials:

```bash
cp .env.example .env.local
```

3. Run the dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
